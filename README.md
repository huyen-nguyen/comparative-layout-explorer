## Development

Install Yarn from https://yarnpkg.com/lang/en/.

Clone the repo:

```bash
git clone git@github.com:sehilyi/comparative-layout-explorer.git
cd comparative-layout-explorer
```

Install the dependencies:

```bash
yarn install
```

Start the test server:

```bash
yarn start
```
